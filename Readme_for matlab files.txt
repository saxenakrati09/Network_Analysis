dijkstras.m- implements dijkstras algorithm and give shortest path between nodes

clustering.m- gives clustering coefficient and average clustering coefficient

characteristicpathlength.m- outputs average path length of the network using adjacancy matrix
in dijkstras.m file

wattsstrogatz.m- generates watts-strogatz network

plotws.m- plots C(p)/C(0) plot

plotL.m- plots L(p)/L(0) plot

BAmodel.m- generates Barabasi-Albert model, shows degree, degree distribution and average clustering
coefficints and characteristic path length of the network

erdosrenyi.m- generates random erdos-renyi graph

edgelist.m- converts adjacancy matrix into edgelist

degree.m- gives degree and degree distribution
